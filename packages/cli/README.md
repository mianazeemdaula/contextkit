# contextkit

Save your AI context once. Inject into any AI tool.

## Install

```bash
npm install -g contextkit
```

## Three commands

```bash
ck init                  # create ~/.contextkit/
ck add work              # open $EDITOR to author a new context
ck copy work             # copy it to the system clipboard
```

Run `ck --help` to see every command.
